# Car-Game


Car game using Pygame.


# Environment
```
OS: Windows10
Python: Python3.10+(have installed necessary dependencies)
```

# Usage
```
Step1:
pip install pygame
Step2:
run "python Car-Game.py"
```

# Controls
```
P Key - Pause
Left Key - Left
Right Key - Right
A Key - Accelarator
B Key - Brake

```
