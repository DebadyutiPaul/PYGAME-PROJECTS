# Tic-Tac-Toe


Tic-Tac-Toe game using Pygame.


# Environment
```
OS: Windows10
Python: Python3.10+(have installed necessary dependencies)
```

# Usage
```
Step1:
pip install -r requirements.txt
Step2:
run "python Tic-Tac-Toe.py"
```
# Controls
```
Mouse keys
```
