# Flappy-Bird

Flappy-Bird game using Pygame.


# Environment
```
OS: Windows10
Python: Python3.10+(have installed necessary dependencies)
```

# Usage
```
Step1:
pip install pygame
Step2:
run "python Flappy-Bird.py"
```

# Controls
```
Tab Key - To control flying

```
