# Ghost Buster


Ghost Buster game using Pygame.


# Environment
```
OS: Windows10
Python: Python3.10+(have installed necessary dependencies)
```

# Usage
```
Step1:
pip install pygame
Step2:
run "python mygame.py"
```

# Controls
```
Mouse keys
```
